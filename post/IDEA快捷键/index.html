工具手册 - IDEA/Pycharm/GoLand/X

2020-01-15 开发工具

本文主要记录常用开发工具使用技巧等

常用功能

Database   用此功能后，什么PLSQL，nacicat可以完全抛弃了，IDEA集成的Database比PLSQL，nacicat等客户端工具好用太多。

CTRL+E   最近打开文件

Ctrl+Shift+R   全局查找替换

Ctrl+X  删除行

Ctrl+D   复制行

CTRL+ALT+B   在方面名上使用此快捷键可直接查看方法的实现，不再需要点过来点过去了

DOUBLE SHIFT  查找所有你想要的，包括类名，方法名，所有相关都可以快速查到（包括jar包中的).
  
Shift+Click   可以关闭文件 

Ctrl+Alt+L  格式化代码

常用插件

  • Key promoter 用来熟悉快捷键，熟悉后丢掉

  • GsonFormat 通过Gson生成代码

  • String Manipulation 模拟数据，生成序列时非常有用

  • Lombok plugin 简化你的实体类，让你不再写get/set方法，还能快速的实现builder模式，以及链式调用方法

快捷键

C --> Ctrl

S --> Shift

M --> Alt

Cmd --> Command
